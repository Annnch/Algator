Integer factorization algorithm decompose a given integer.

Input: An integer.
Output: An array of integers, which multipy into input integer.
Test of correctness:     The execution of an algorithm for a given integer is correct if \( \displaystyle \prod_{i = 1}^{n} a[i] \); where \(a\) is an output array and \(n\) length of the array \(a\).
Implementation: To implement a integer factorization algorithm the author has to write a java class extending the IntegerFactorizationAbsAlgorithm class
