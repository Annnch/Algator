Trial division is the most basic integer factorization algorithm. In order to decompose an input number \(n\), the algorithm tries if numbers between 2 and \(n\) are divisors of \(n\). When algorithm finds such number, it divides n with that number and continues recursively.

In our implementation, the algorithm returns solution when it finds the first divisor and it goes to \(\sqrt{n}\). Returned result is an array with divisors d and n/d.

Simple pseudocode:

  1. function TrialDivision(\(N\))
  2.   for \(s\) from \(2\) to \(\sqrt{N}\)
  3.    if \(s\) divides \(N\) then
  4.     return \(s, N/s\)
