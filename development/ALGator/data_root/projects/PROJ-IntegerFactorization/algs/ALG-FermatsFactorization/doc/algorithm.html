Fermat's factorization algorithm works simillar to the trial division algorithm, but in different direction. We denote with \(n\) the number we wand to factorize. Trial division algorithm starts with 2 and stops with a number smaller or equal to \(\sqrt{n}\). On the other hand Fermat's algorithm starts with \(\sqrt{n}\), therefore algorithm works fast for the numbers that are a product of two numbers close together.

Let \(n = a \cdot b\), where \(a\) and \(b\) are integers. We can also write \(n\) as \( n = x^2 - y^2 = (x - y)(x + y) \), where \( x = \frac{a + b}{2} \text{ and } y = \frac{a - b}{2} \). Fermat's algorithm now follows next steps:

  1. \(x = \sqrt{n}\) (rounded up)
  2. \(y = x^2 - n\)
  3. if \(y\) is a square of some number, we are done.
  4. if \(y\) is not a square of some number, we increase \(x\) by 1 and calculate \(y\) as before.

Time complexity: \( O(n) \).
