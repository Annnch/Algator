Integer factorization algorithm decompose a given integer.

Input: A big integer.
Output: A array of integers, which multipy into input integer.
Test of correctness:     The execution of an algorithm for a given big integer is correct if \( \displaystyle \prod_{i = 1}^{n} a[i] \); where \(a\) is an output array and \(n\) number in the array.
